# 监控设计文档

## 监控指标

### 系统指标
- CPU 使用率
- 内存使用率
- 磁盘使用率
- 网络流量

### 应用指标
- QPS
- 响应时间（P50/P95/P99）
- 错误率
- 超时率

### 业务指标
- 在线用户数
- 注册用户数
- 帖子发布数
- 组队成功数

## 告警规则

### 紧急告警
- CPU > 90% 持续 5 分钟
- 内存 > 95%
- 错误率 > 5%
- 服务不可用

### 警告告警
- CPU > 80% 持续 10 分钟
- 内存 > 85%
- API 响应时间 > 1s

## 监控工具
- Prometheus: 指标收集
- Grafana: 可视化
- Loki: 日志聚合
