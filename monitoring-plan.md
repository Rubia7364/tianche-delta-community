# 监控方案文档

## 监控指标

### 系统指标

**服务器监控：**
- CPU 使用率
- 内存使用率
- 磁盘使用率
- 磁盘 I/O
- 网络流量
- 网络连接数

**数据库监控：**
- 连接数
- 查询 QPS
- 慢查询数量
- 锁等待时间
- 缓冲池命中率

**Redis 监控：**
- 内存使用率
- 命中率
- 连接数
- 操作 QPS

### 应用指标

**API 性能：**
- 请求 QPS
- 响应时间（P50/P95/P99）
- 错误率
- 超时率

**业务指标：**
- 在线用户数
- 注册用户数
- 帖子发布数
- 组队成功数
- 订单数量
- 支付成功率

### 埋点指标

**用户行为：**
- 页面访问量（PV）
- 独立访客数（UV）
- 用户停留时间
- 页面跳出率
- 按钮点击率

**功能使用：**
- 发帖数量
- 评论数量
- 组队发起数
- 组队加入数
- 搜索次数

## 告警规则

### 紧急告警（立即处理）

- CPU > 90% 持续 5 分钟
- 内存 > 95%
- 磁盘 > 90%
- 数据库连接数 > 90%
- 错误率 > 5%
- 服务不可用

### 警告告警（1小时内处理）

- CPU > 80% 持续 10 分钟
- 内存 > 85%
- 磁盘 > 80%
- API 响应时间 > 1s
- 错误率 > 1%
- 慢查询 > 100 次/分钟

### 提示告警（24小时内处理）

- CPU > 70% 持续 30 分钟
- 内存 > 75%
- 磁盘 > 70%
- 缓存命中率 < 70%

## 告警通知

### 通知渠道

- **邮件**：所有告警
- **短信**：紧急告警
- **飞书/钉钉**：所有告警
- **电话**：服务不可用

### 通知对象

- **开发团队**：应用告警
- **运维团队**：系统告警
- **值班人员**：紧急告警

## 日志策略

### 日志级别

- **ERROR**：错误日志（必须记录）
- **WARN**：警告日志
- **INFO**：信息日志
- **DEBUG**：调试日志（仅开发环境）

### 日志内容

**应用日志：**
- 时间戳
- 日志级别
- 请求 ID
- 用户 ID
- 接口路径
- 错误堆栈

**访问日志：**
- 时间戳
- 客户端 IP
- 请求方法
- 请求路径
- 响应状态码
- 响应时间

**审计日志：**
- 时间戳
- 用户 ID
- 操作类型
- 操作对象
- 操作结果

### 日志保留

- **应用日志**：7 天
- **访问日志**：30 天
- **审计日志**：180 天
- **错误日志**：90 天

## 监控工具

- **Prometheus**：指标收集
- **Grafana**：可视化看板
- **Loki**：日志聚合
- **Alertmanager**：告警管理
